# ebitris

A Tetris clone built with Go and [Ebitengine](https://ebitengine.org/), created to learn the basics of game design.

## Commands

| Command      | Description       |
|--------------|-------------------|
| `make build` | Build the binary  |
| `make test`  | Run all tests     |
| `make lint`  | Run linter        |
| `make help`  | Show all commands |
